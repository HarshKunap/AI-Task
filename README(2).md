# Uninformed Search Algorithms

## Implemented Algorithms
- Breadth First Search (BFS)
- Depth First Search (DFS)

## Problem Chosen
Missionaries and Cannibals

## State Representation
(M_left, C_left, Boat)

## Goal
Move all missionaries and cannibals safely to right bank.

## How to Run
python performance.py

## Comparison

BFS:
- Complete
- Optimal
- Higher memory usage

DFS:
- Not always optimal
- Lower memory usage
- May explore deep wrong paths
