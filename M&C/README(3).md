# Turing Test Implementation

## Concept
Proposed by Alan Turing in 1950, the Turing Test evaluates whether a machine can exhibit intelligent behavior indistinguishable from a human.

## Components
- Interrogator (Judge)
- Human Participant
- AI Agent
- Logging Module
- Evaluation Module

## How to Run
1. Navigate to folder
2. Run:
   python server.py

## Features
- Blind AI randomization
- Conversation logging
- Judge evaluation
- Accuracy scoring
